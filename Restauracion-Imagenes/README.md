# Restauracion-Imagenes

Aplicación para restauración y mejora de imágenes usando Stable Diffusion, ControlNet, IP-Adapter/InstantID y análisis visual con Gemini.

## Estructura
- `app.py`: Aplicación principal Streamlit
- `models/`: Procesamiento y análisis
- `utils/`: Utilidades de imagen y UI
- `assets/ejemplos/`: Imágenes de ejemplo
- `config/config.yaml`: Configuración

## Instalación
Ver `requirements.txt` para dependencias.
